# 🌙 SeoulAfterDark-Game🎮

Красивая игра в сеттинге ночного Сеула, написанная на Python с использованием библиотеки **Pygame**. Проект разработан для защиты персональных проектов.

## 🌟 О проекте

Вы — владелец небольшой уютной лавки уличной еды в самом сердце ночного Сеула. Ваши главные клиенты — культовые персонажи известных корейских дорам. Успейте накормить их любимыми блюдами, пока они не потеряли терпение!

### 👥 Дорамы и персонажи в игре
На данный момент в игру заглядывают герои триллера **«Незнакомцы из ада» (Strangers from Hell)**:
* 🩺 **Со Мун Джо** (Seo Moon-jo)
* ✍️ **Юн Чон У** (Yoon Jong-woo)

---

## 🍳 Игровой процесс (Геймплей)

На нижней панели (рабочем столе) расположены ингредиенты. Клиент делает заказ, а ваша задача — быстро собрать правильное блюдо по рецепту.

### 📜 Доступное меню:
1. **Кимпаб (Gimbap)** 🌯 — Рецепт: `Ингредиент 1` + `Ингредиент 2` + `Ингредиент 3`
2. **Пибимпаб (Bibimbap)** 🍲 — Рецепт: `Ингредиент 4` + `Ингредиент 5` + `Ингредиент 3`

### ⚠️ Правила игры:
* Время на выполнение заказа ограничено таймером.
* Если вы не успеваете приготовить блюдо, рассерженный клиент уходит.
* **Игра завершается**, если вы упустите **3 клиентов**.

---

## 🚀 Как запустить игру на своем ПК

1. **Установите Python** версии 3.8 или выше (если еще не установлен).
2. **Установите библиотеку Pygame**. Для этого откройте терминал (командную строку) и выполните:
   ```bash
   pip install pygame
   ```
3. **Скачайте этот репозиторий** и запустите главный файл:
   ```bash
   python game.py
   ```
